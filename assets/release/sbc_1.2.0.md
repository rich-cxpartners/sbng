#### About this release:

#### 1. How to install it

```bash
npm install @sb/components@1.2.0
```

#### 2. Changes

##### Angular 9
This library has been updated to Angular 9

##### Form-item
Some classes have been added inside form item to help project to custom some éléments if they need to.
`sbf-formitem-required` has been changed to `sbf-form-item-required` to be more homogeneous.

##### Utility classes
Some utility classes have been added to help you to structure your pages.
You can see some example here: [CSS utility classes page](http://slnxta01.marc.fr.ssg:5555/#/showcase/css-utility)
